# 量子重叠计算工具开发计划

## 一、项目概述
根据 `core_feature.md` 文档，实现一个量子重叠计算工具，包含速算版本和精算版本，使用 Ionic UI 组件库结合 Vue 3 (TypeScript, Composition API with setup) 技术栈。

## 二、页面结构规划

### 1. 导航菜单设计
- 重新设计侧边菜单，包含以下导航项：
  - 首页/速算版本
  - 精算版本（订阅）
  - 关于我们
  - 隐私政策
  - 条款与条件
  - 反馈
  - 赞助

### 2. 核心页面实现

#### 2.1 速算版本页面 (`QuickCalcPage.vue`)
- 输入表单：
  - 人体 A：位置坐标、体重、身高、出生日期
  - 人体 B：位置坐标、体重、身高、出生日期
- 计算结果展示：
  - 累计量子重叠数量
  - 简单的分享功能

#### 2.2 精算版本页面 (`PreciseCalcPage.vue`)
- 输入表单：
  - A/B：出生日期、历史位置/坐标、体重、身高
- 计算结果展示：
  - 不同分类（量子、传播方式）的量子扩散抵达率
  - 量子重叠数量和与个体总量子数百分比
  - 可定制的分享页面

#### 2.3 关于页面 (`AboutPage.vue`)
- 项目介绍
- 联系作者
- 开源协议

#### 2.4 隐私政策页面 (`PrivacyPolicyPage.vue`)
- 隐私政策内容

#### 2.5 条款与条件页面 (`TermsPage.vue`)
- 条款与条件内容

#### 2.6 反馈页面 (`FeedbackPage.vue`)
- 反馈表单

#### 2.7 赞助页面 (`SponsorPage.vue`)
- 赞助人员列表
- 订阅入口

## 三、核心功能实现

### 1. 速算版本算法
- 实现文档中描述的速算版本算法：
  - 计算 A/B 体内的量子数量
  - 计算共生时间
  - 计算平均距离
  - 估算分钟级量子交换数量
  - 计算直接、间接、跃迁量子交换量
  - 输出累计量子重叠数量

### 2. 精算版本算法
- 实现文档中描述的精算版本算法：
  - 量子分类（水分子、氧、其它）
  - 传播方式（呼吸、蒸发、热运动、外力、量子跃迁）
  - 各种扩散方式的占比
  - 各种量子在不同传播方式下的占比
  - 总扩散量或各种量子的扩散量
  - 不同传播方式的扩散抵达率

### 3. 分享功能
- 实现简单的分享页面（速算版本）
- 实现可定制的分享页面（精算版本）：
  - 结果选择
  - 配色选择

## 四、技术实现原则

1. **优先使用 Ionic 原生组件和样式**
   - 使用 `ion-input`、`ion-select`、`ion-button` 等原生组件
   - 使用 Ionic 提供的布局组件如 `ion-grid`、`ion-row`、`ion-col`

2. **最大限度减少自定义 CSS**
   - 仅在必要时使用自定义样式
   - 优先使用 Ionic 提供的 CSS 变量进行调整

3. **统一的样式管理**
   - 若 Ionic 变量无法满足需求，在 `src/theme/variables.css` 中定义新的 CSS 变量
   - 确保样式的一致性和可维护性

4. **TypeScript 类型安全**
   - 为所有数据和函数添加类型定义
   - 使用 Composition API with setup 语法
