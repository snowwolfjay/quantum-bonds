# 量子联结计算器 核心功能

> 这是一款计算两个人之间，量子重叠的工具。
> 所谓量子重叠：A 体内的量子，来自 B 体内的数量。
> 核心目标：展现人与人与自然之间的量子重叠，认识到人与自然的一体化，实现人与人、自然之间的和谐共生，最终消除极端不平等、阶级，实现和平。

模型简化说明： 计算 A 来自 B 体内的量子数量，仅考虑 A 的扩散到外界进入 B 体内的过程，反之类似。

## 核心算法

### 速算版本

#### 输入参数

人体 A： 所在位置坐标、体重、身高、出生日期（年龄）
人体 B： 所在位置坐标、体重、身高、出生日期（年龄）

#### 计算方法：

1. 计算 A 体内的量子数量：A 体重 _ A 身高 _ 系数
2. 计算 B 体内的量子数量：B 体重 _ B 身高 _ 系数
3. 计算 A 和 B 的共生时间，分钟级别： 也就是二者年龄最小值；
4. 计算 A 和 B 的平均距离，根据当前所在位置，计算二者的平均距离，单位：米。
5. 估算出 A 的分钟级量子交换数量；
6. 计算 A 与 B 的直接量子交换量、间接量子交换量、跃迁量子交换量。
   - 直接量子交换量：A 体内的量子，直接流向 B 体内的数量 - 基于热运动和速度。
     = 直接量子交换量系数 \* 量子交换速率 （与平均距离成反比）
   - 间接量子交换量：A 体内的量子，通过其他中间体，流向 B 体内的数量 - 基于气流和其它运动物体。
     = 间接量子交换量系数 \* 量子交换速率 （与人口密度成反比）
   - 跃迁量子交换量：A 体内的量子，通过量子跃迁，流向 B 体内的数量 - 亚光速级别量子的传输量。
     = 跃迁量子交换量系数 \* 量子交换速率 （与距离成正比）

输出： 累计量子重叠数量

### 精算版本

#### 输入参数

A/B: 出生日期（年龄）,历史位置/坐标,体重,身高 （暂时只支持一个）

#### 计算方法：

量子分类：

1. 水分子
2. 氧
3. 其它

传播方式：

1. 呼吸
2. 蒸发
3. 热运动
4. 外力
5. 量子跃迁

#### 任务

1. 获取各种扩散方式的占比

2. 获取各种量子在不同传播方式下的占比

3. 总扩散量或各种量子的扩散量（求和）

4. 确定不同传播方式的扩散抵达率
   > 扩散抵达率，是一个和距离相关的参数或者函数，指代 A 扩散到环境的量子抵达 B 体内的概率/比例
